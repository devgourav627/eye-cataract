# 👁️ Eye Cataract Detection using Machine Learning

## 📌 Project Overview

This project focuses on detecting **cataracts in eye images** using
**machine learning and computer vision techniques**.

Cataracts are a common eye condition that causes clouding of the eye
lens and may lead to vision impairment if not detected early. This
project demonstrates how **AI and image processing** can assist in
identifying cataract conditions from eye images.

The system processes eye images and predicts whether the eye is **normal
or affected by cataract**.

------------------------------------------------------------------------

# 🎯 Objectives

-   Detect cataracts from eye images
-   Apply image preprocessing techniques
-   Train a machine learning model for classification
-   Demonstrate the use of **AI in medical image analysis**
